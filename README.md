# app
新番下载工具，基于aira2和miobt，acgrip等种子站，目前仅在windows进行了测试
下载番剧时会根据番剧名称自动创建文件夹
![image text](https://raw.githubusercontent.com/LinneRELLa/ROT/master/imgs/readme4.png)
__因种子站被墙，下载番剧前需要在修改aria2/start.bat最后的--all-proxy选项__
